## Как опубликовать через GitHub Pages

1. В репозитории откройте **Settings → Pages**
2. В разделе **Build and deployment** выберите Source: **Deploy from a branch**
3. Branch: `main`, папка `/ (root)` → **Save**
4. Через 1–2 минуты сайт будет доступен по адресу `https://ВАШ_ЛОГИН.github.io/abadansport/`
